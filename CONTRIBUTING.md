# Contributing

## Setup
- Clone, create `.env.local` from `.env.example`, install deps
- Run: `npm run lint`, `npm run typecheck`, `npm run test`

## Branches & Commits
- Branches: `feature/…`, `fix/…`, `chore/…`
- Conventional Commits: `feat: …`, `fix: …`, `docs: …`

## PR Checklist
- [ ] Lint and typecheck pass
- [ ] Unit tests added/updated; coverage respected
- [ ] E2E updated if user flow changes
- [ ] Docs updated (README or docs/* as needed)

## Reviews
- Keep changes small and focused; describe “why”
